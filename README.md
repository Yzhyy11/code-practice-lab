# code-practice-lab

这是我的个人代码练习仓库，主要用来长期练习 C++、Python、算法、数据结构和机器学习。

## 目录结构
- `cpp/`：存放 C++ 相关练习  
  - `basics/`：C++ 基础示例  
  - `algorithms/`：算法实现  
  - `data_structures/`：数据结构实现  
  - `projects/`：综合项目  
  - `CMakeLists.txt`：C++ 工程的 CMake 配置
- `python/`：存放 Python 相关练习  
  - `basics/`：Python 基础示例  
  - `ml/`：机器学实验  
  - `data_analysis/`：数据分析脚本  
  - `scripts/`：其他工具脚本
- `leetcode/`：LeetCode 题解  
  - `cpp/`：C++ 题解  
  - `python/`：Python 题解
- `notes/`：学习笔记  
  - `cpp_notes.md`：C++ 学习笔记  
  - `python_notes.md`：Python 学习笔记  
  - `algorithm_notes.md`：算法学习笔记
